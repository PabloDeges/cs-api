The Endpoint for this API is: localhost:8000
GET
/api/<skin_name>
retrieve skin Name,price in USD(Field Tested) and skin image for the specific skin. Example: /api/AK-47_Redline
GET
/api/random
retrieve skin info for a random skin. Example: /api/random
GET
/api/randomclass/<class_name>
retrieve skin info for a random skin of a specific class. Example: /api/randomclass/Pistol
GET
/api/random/<weapon_name>
retrieve skin info for a random skin of a specific weapon. Example: /api/random/SSG-08
